視線で辞書を引くシステムです.
未完成品なので, 視線をトレースするモジュールと
辞書を制御するモジュールが別々になっています.
これは今後統合していきます.
現在はモジュールを同時に起動することで, 視線で辞書を引くことが可能です.


-使い方----------------------------------------------
・EyeXMouse-masterを起動し
gaze.exeを起動するとマウスポインタで辞書を引く機能機能を搭載したプログラムが動きます.　
・textフォルダ内に文章を入れてあります.
この文章は":"で本文と注釈を区切っています. 任意の文章を埋め込むことが可能です.
